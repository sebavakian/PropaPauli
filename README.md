# PropaPauli

PropaPauli is a Aleph based library that provides a framework for doing Pauli propagation in many contexts. 
This library is intended to be easy to integrate into other alpeh based quantum workflows. 

This implementation is currently uses a sparse symbolic representation of pauli strings that can be used with other numeircal methods in the Workshop. 

## Installation

All you need to do to use the code is load the repo into your workspace on the Workshop and include the files you need with `include "path/to/file.aleph"`. 

## Getting started

Once you have loaded all the files into the your workspace, open the example markdown files. It contains runable code that you can follow to learn how to use this library.
